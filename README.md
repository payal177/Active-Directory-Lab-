# Enterprise Active Directory Lab


## Project Overview

This project demonstrates the implementation of a complete Enterprise Active Directory environment using Windows Server 2022 and VMware Workstation.

The lab was designed to simulate real-world Windows Server administration tasks commonly performed by System Administrators, Windows Administrators, and IT Infrastructure Engineers.

The project covers the complete lifecycle of Active Directory deployment, including domain creation, user and group management, DNS, DHCP, Group Policy, File Server configuration, Home Folder mapping, PowerShell administration, and client domain integration.

Throughout this lab, real-world administrative scenarios such as password resets, user onboarding, user offboarding, department transfers, file sharing, and troubleshooting were also performed.
---

## Project Objectives

The primary objectives of this project were to:

- Build a complete Active Directory Domain Services (AD DS) environment.
- Deploy and manage Windows Server 2022 in VMware Workstation.
- Configure DNS and DHCP services.
- Create and manage Organizational Units (OUs), users, and security groups.
- Implement Group Policy Objects (GPOs).
- Configure File Server and NTFS permissions.
- Configure user Home Folder mapping.
- Join Windows 10 clients to the domain.
- Perform common Windows Administrator troubleshooting tasks.
- Learn PowerShell for Active Directory administration.
- Document the entire implementation for interview preparation and portfolio purposes.
