# 🚨 Scenario-Based Questions

### Q1. A user cannot log in. What will you check?

- Verify whether the account is disabled.
- Check if the account is locked.
- Reset password if required.
- Verify domain connectivity.
- Check DNS.
- Verify Group Policy.

---

### Q2. User cannot access the shared folder.

Steps:

- Verify network connectivity.
- Check Share Permissions.
- Check NTFS Permissions.
- Verify Security Group membership.
- Ask the user to log off and log in again.

---

### Q3. A new employee joins the company.

Steps:

- Create User Account.
- Add to Organizational Unit.
- Add Security Groups.
- Configure Home Folder.
- Provide domain credentials.

---

### Q4. Employee leaves the company.

Steps:

- Disable User Account.
- Remove Security Group Membership.
- Archive Home Folder.
- Document the changes.

---
