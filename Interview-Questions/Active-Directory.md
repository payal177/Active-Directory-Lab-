# 👤 Active Directory

### Q1. What is Active Directory?

**Answer:**Active Directory (AD DS) is Microsoft's centralized directory service used to manage users, computers, groups, and security policies within a Windows domain.
---

### Q2. What is a Domain Controller?

**Answer:**A Domain Controller (DC) is a Windows Server that authenticates users, stores Active Directory data, and manages security policies.

---

### Q3. What is an Organizational Unit (OU)?

**Answer:**An Organizational Unit is a logical container used to organize users, computers, and groups for easier administration and Group Policy management.

---

### Q4. Difference between Security Group and Distribution Group?

**Answer:**

- Security Groups are used for permissions.
- Distribution Groups are used only for email distribution.
